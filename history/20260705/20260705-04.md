# Kotlin 2.3.0 업그레이드 + JVM 25 바이트코드 타깃

- **일시**: 2026-07-05
- **커밋**: chore[build] : Kotlin 2.3.0 업그레이드 + JVM 25 바이트코드 타깃

## 작업 내용
Kotlin 2.2.20 에서 2.3.0 으로 업그레이드했다(JDK 25 를 지원하는 버전). 2.4.0 은 Spring Boot 4.0 궁합이 아직 애매해 보류했다. 2.3.0 부터 JVM_25 타깃을 지원하므로 기존 24 폴백(options.release=24, JvmTarget.JVM_24)을 25 로 통일했다. 전체 빌드(gradlew build) 성공을 확인했다.

## 변경 파일
- backend/gradle/libs.versions.toml — kotlin 2.3.0 (+ 2.4.0 보류 사유 주석, jjwt 주석 발급 주체 user 로)
- backend/build.gradle.kts — JavaCompile release 25, KotlinCompile jvmTarget JVM_25

## 주요 결정사항
2.4.0 미채택(Spring Boot 4.0 호환 애매).
